# zebr-frontend-cli

前端脚手架，快速创建项目模版，提高前期开发效率。

## 功能

- 快速创建项目模版
- 在编辑器打开项目并启动本地服务

## 提示帮助

```sh
zebr-cli
```

- 获取提示，展示功能命令

## 创建项目

```sh
zebr-cli create <name>
```

- 命令行窗口进入目录地址，输入执行
- 根据提示选择项目模版（vue3+Ts、react等）
- 通过git仓库地址拷贝项目

## 在编辑器打开项目并启动本地服务

```sh
zebr-cli open
```

- 命令行窗口进入项目地址
- 自动在编辑器打开项目
- 检测是否已初始化项目依赖
- （否）自动或者手动选择包管理工具进行初始化安装依赖
- （是）自动开启本地服务
